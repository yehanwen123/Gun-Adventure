# 开发环境配置指南

## 环境要求

- **Node.js**：v14 或更高版本（[下载地址](https://nodejs.org/)）
- **现代浏览器**：Chrome、Firefox 或 Edge 最新版
- **代码编辑器**：推荐 [Visual Studio Code](https://code.visualstudio.com/)
