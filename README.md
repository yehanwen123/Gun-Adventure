## 📖 项目概述

《Gun Adventure》是一款2D俯视角射击冒险游戏，创新性地融合了经典桌游DND（龙与地下城）的数值策略与像素风Roguelike的爽快射击玩法。玩家将扮演冒险者，在多个精心设计的关卡中探索、战斗并挑战Boss。游戏采用复古像素风格，操作直观流畅，并包含轻度Roguelike元素（角色成长、技能组合）。

**核心特色**：引入“骰子检定”机制——在关键时刻（如开宝箱、触发陷阱、使用技能、Boss战）通过虚拟骰子掷出结果决定行动成败、伤害倍率或特殊效果，为冒险增添策略深度和戏剧性。常规操作（移动、射击、切换武器）则保持流畅的即时战斗体验。

---

## ✨ 功能特性

- **多职业选择**：提供战士、游侠、法师等职业，各具独特战斗风格。
- **随机地牢**：每局游戏随机生成房间、走廊及敌人分布，保证高重复可玩性。
- **D20骰子检定**：在战斗中实时掷骰决定命中、暴击、闪避；开宝箱、触发陷阱等事件也受骰子影响。
- **角色成长**：击败敌人获得经验，升级解锁新技能，技能效果由骰子检定决定强度。
- **Boss战**：每个关底Boss拥有多阶段攻击，并与骰子机制深度结合。
- **战斗日志**：实时显示每次攻击的骰子结果及判定（命中/未命中/暴击）。
- **像素美术风格**：320x180分辨率，强制像素化渲染，保持复古质感。
- **自动存档**：支持本地存档及断线重连（在线模式）。

## 🛠 技术栈

- **前端**：HTML5 + Canvas + JavaScript (ES6)
- **后端**：Node.js + Express (轻量，用于数据存储和多人扩展)
- **数据库**：本地存储 (localStorage) + 云端可选 (MongoDB / 文件存储)
- **构建工具**：无（原生JS），或使用Parcel/Webpack（可选）
- **版本控制**：Git + GitHub
- **部署**：GitHub Pages / Vercel（前端），Heroku / Render（后端）

---

## 🧩 系统架构

项目采用模块化设计，主要组件如下：

- **用户交互模块**：处理键盘/鼠标输入，显示UI（职业选择、HUD、骰子动画、战斗日志）。
- **核心游戏逻辑模块**：管理游戏流程（关卡切换、胜利/失败判定、角色成长）、AI逻辑、职业行为。
- **D20骰子检定引擎**：模拟20面骰子，执行命中公式（1d20 + 属性加值 ≥ 敌人AC），处理各类事件的骰子规则。
- **随机地图生成模块**：使用随机算法生成连通的房间和走廊，分布敌人、宝箱、陷阱，具备自动重试机制。
- **资源管理模块**：预加载像素资源、动画、音效，动态释放内存。
- **渲染与表现模块**：负责像素渲染、动画同步、音效反馈。
- **存储与适配模块**：管理本地存档，根据设备性能自动调整特效以保持60FPS。

## 🚀 快速开始

### 环境要求

- 现代浏览器（Chrome, Firefox, Edge最新版）
- Node.js v14+（如需运行后端）

### 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-team/gun-adventure.git
   cd gun-adventure
   ```

2. 安装依赖（仅后端需要）：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   - 前端：直接打开 `index.html` 或使用Live Server。
   - 后端：`npm start`（默认端口3000）。

4. 访问 `http://localhost:3000` 开始游戏。
